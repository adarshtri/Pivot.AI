description: This workflow helps create learning material for Rust. The learning material will be created as the project pivot-ai-cli progresses. The concepts used in pivot-ai-cli project will be referenced to identify topics not in learning and update learning.
---

Rust Learning Doc Locations - rust-learning/ under project root.
Rust Learning Doc Nameing - rust-learning/{topic-number}-{topic-name}.md

**Don't update project files ever.**
**Only read project files to figure out which topics are not in Rust learning.**
**No 2 files have same {topic-number}.**
**{topic-numbers} must be integers. No x.y type naming.**
